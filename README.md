Sales Dashboard - Power BI

#Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance across different regions, categories, customer segments, and payment methods. It helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

#Business Problem

A retail business needed a dashboard to monitor sales, profit, customer segments, and regional performance in one place to support better business decisions.

# Dashboard Features

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Sales
- Regional Analysis
- Category Performance
- Customer Segment Analysis
- Payment Mode Analysis
- State-wise Sales Map
- Interactive Region Filter

#Tools Used

- Power BI
- DAX
- Microsoft Excel

#Key Insights

- Consumer segment contributes the highest sales.
- Sales increase significantly during the last quarter of the year.
- Regional analysis highlights top-performing markets.
- Interactive filters allow users to explore data dynamically.
